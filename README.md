# queueing-theory-demo
Queueing Theory Demo

Instructions:
```
git clone https://github.com/YogevKr/queueing-theory-demo.git
cd queueing-theory-demo
docker-compose up
```

Go to your browser, you should see the Streamlit app running on http://localhost:8501.
